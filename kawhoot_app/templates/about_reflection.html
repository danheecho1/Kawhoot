Reflection & Future Plans

          For Kawhoot, I used the same languages, frameworks, and DB I used for Myspice: HTML, CSS, minimal JS, Python with Flask and Jinja, and MySQL. This is one of the biggest regrets I have for this project - I wish I tried Django instead of Flask, and maybe React for the frontend. My next project will definitely use different frameworks. Getting an API involved is also on the list.
          Another thing I did not realize until now (which is embarrassing) is that I did not make this application to be responsive or mobile friendly. I came up with Kawhoot's design thinking I would make it fit within the viewport so that there would be no need for scrolling.
